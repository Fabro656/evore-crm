VALIDACIONES

  ✓ nombre único (no duplicados)
  ✓ costo_interno >= 0
  ✓ precio_venta >= 0
  ✓ unidad es select (no libre)
  ✓ categoria max 100


LÓGICA ESPECIAL
===============

CALCULADORA EMPAQUES:
  Input: alto_prod, ancho_prod, largo_prod, peso_unitario, peso_max_caja

  1. max_unidades = floor(peso_max_caja / peso_unitario)
  2. opciones = [6, 12, 24, 48] filtradas si <= max_unidades
  3. Para cada opción:
     - alto_caja = alto_prod + 2
     - ancho_caja = ancho_prod + 2
     - largo_caja = (largo_prod × unidades) + 2
     - peso_total = unidades × peso_unitario
  4. Retorna max 3 opciones (JSON)

AL APROBAR EMPAQUE:
  1. Crea MateriaPrima:
     - nombre: 'Caja {producto.nombre} x{unidades}'
     - categoria: 'empaques'
     - unidad: 'unidades'
     - stock_disponible: 0
     - activo: true
  2. Asigna: empaque.materia_prima_id = mp.id
  3. Marca: empaque.aprobado = True
  4. Commit y flash success

MARGEN SERVICIOS:
  margen = (precio_venta - costo_interno) / precio_venta * 100

  Badges por margen:
  - < 20%:  warning (bajo)
  - 20-40%: info (moderado)
  - > 40%:  success (saludable)

PREVISUALIZACIÓN EN TIEMPO REAL:
  JavaScript: actualizarMargen()
  - Actualiza on input/change en precio y costo
  - Muestra valores formateados y margen %
  - Badge dinámico
